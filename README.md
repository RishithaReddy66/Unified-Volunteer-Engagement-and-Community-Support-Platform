# Unified Volunteer Engagement and Community Support Platform

## 📖 Overview

The Unified Volunteer Engagement and Community Support Platform is a full-stack web application designed to simplify collaboration between Non-Governmental Organizations (NGOs) and volunteers. It provides a centralized platform where NGOs can create and manage community service events, while volunteers can discover, register, and participate in social activities.

The platform improves communication, transparency, and efficiency in volunteer management while encouraging greater community participation.

---

## ✨ Features

- User Registration and Login
- NGO Event Creation and Management
- Volunteer Event Registration
- Community Support Services
- Event Tracking
- User Profile Management
- Responsive User Interface
